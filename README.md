# ACP-Bench

ACP-Bench is a benchmark for evaluating agentic harness systems in autonomous computational pathology workflows. It focuses on whether agentic systems can convert pathology analysis goals into executable, traceable and clinically bounded workflows, rather than only producing plausible final text.

This repository is being prepared for progressive public release. Code, benchmark metadata and releasable derived data will be added in stages after de-identification, license review and removal of restricted data references.

## Scope

ACP-Bench evaluates pathology workflow tasks across biomarker, morphology and prognosis settings. The benchmark combines workflow execution assessment, diagnostic-result normalization, clinical-boundary alignment, safety-boundary review and pathologist validation.

The manuscript reports evaluations across multiple agentic harness systems and model backends. The repository will provide the materials needed to inspect, reproduce and extend the benchmark where data-use terms allow.

## Planned Releases

The public release will be staged. Planned materials include:

- Task metadata and workflow-requirement mapping tables
- Expert-adjudicated workflow scores
- Diagnostic-result summaries and normalization code
- Clinical workflow alignment and safety-boundary review outputs
- Pathologist validation summaries
- Reflection summaries and long-horizon tier assignments
- Source data for quantitative figures
- Analysis and figure-generation scripts

Some whole-slide image derivatives, model-attention inputs and source materials may remain restricted because they are governed by external data-use terms. When a file cannot be released, the repository will document the restriction and, where possible, provide derived summaries or reproduction instructions.

## Repository Status

Initial public release is in progress. Until the release is complete, this repository should be treated as the canonical landing page for ACP-Bench rather than a complete reproduction package.
